General Questions

Vidi is a powerful media player that enhances your streaming experience by integrating with various addons and debrid services. It provides a seamless way to access and manage high-quality streaming sources. Check out our addons page to explore compatible addons.

Yes, Vidi requires either a Real-Debrid or AllDebrid subscription to function. These services provide high-speed, reliable access to premium streaming sources. You can sign up for either service through their respective websites.

Setup & Configuration

Setting up Vidi with addons is a simple process:

  1. Install Vidi if you haven't already
  2. Get your API key from your debrid service (Real-Debrid or AllDebrid)
  3. Visit our addons page to configure compatible addons
  4. Use the configuration tool to set up your preferred options
  5. Click the installation button to add the addon to Vidi

Features & Functionality

When configuring addons in Vidi, you can select from various quality options:

  • 4K/2160p content
  • 1080p Full HD
  • 720p HD
  • HDR content
  • Dolby Vision
  • Remux quality

You can customize which quality options you want to see in the addon configuration settings.

Vidi works with popular addons like Torrentio and Comet. Visit our addons page to see the full list of compatible addons and their configuration options. Each addon can be customized with your preferred settings for the best streaming experience.
